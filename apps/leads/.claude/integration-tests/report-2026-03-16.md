## Architecture notes

- **No Composio dependency** — All integrations use direct API calls. The prompt's references to "Via Composio" are not applicable to LeadSens.
- **Registry-driven** — All 57 connectors defined in `src/server/lib/integrations/registry.ts` with inline `testConnection` functions.
- **Dynamic routing** — `src/app/api/integrations/[tool]/route.ts` handles API key connect/disconnect for all tools.
- **OAuth handling** — `src/server/lib/integrations/oauth-handler.ts` for OAuth flows (HubSpot, Salesforce, Calendly, Slack, Outreach, ZoomInfo).
- **Encryption** — All stored API keys encrypted AES-256-GCM via `src/lib/encryption.ts`.
