# Competitor Landing Page Analysis

> Generated from plan Phase 0. Update with Playwright research for live data.

## Cold Email Tools

### Instantly.ai
- **Headline:** "Turn Leads into Clients"
- **Hero:** Clean, product screenshot, immediate CTA
- **Sections:** Social proof bar (logos), How it works (3 steps), Features grid, Testimonials, Pricing
- **Colors:** Blue dominant (#2563EB), white bg, minimal gradients
- **Pattern:** Product-first — show the tool immediately

### Smartlead.ai
- **Headline:** "Scale Cold Emails That Deliver"
- **Hero:** Video demo embed, badge "G2 Top Rated"
- **Sections:** Stats bar, Feature deep-dives, Integrations, Testimonials, FAQ
- **Colors:** Purple-blue gradient, dark mode default
- **Pattern:** Social proof heavy — G2 badges, customer logos, metrics

### Lemlist.com
- **Headline:** "Send cold emails people love"
- **Hero:** Animated product preview, warm personality
- **Sections:** Personalization demo, Multichannel, Results, Pricing
- **Colors:** Yellow accent, warm palette, playful illustrations
- **Pattern:** Personality-driven — brand voice > product features

### Apollo.io
- **Headline:** "Your all-in-one sales intelligence platform"
- **Hero:** Dense value prop, multiple CTAs
- **Sections:** Database stats, Workflow builder, Integrations, Enterprise trust
- **Colors:** Blue/purple, enterprise feel
- **Pattern:** Data-first — lead with database size (275M+ contacts)

## Design References

### Linear.app
- **Notable:** Minimal, typography-focused, dark/light mode excellence
- **Patterns:** Glass cards, subtle gradients, keyboard shortcuts showcase
- **Takeaway:** Restraint is premium

### Vercel.com
- **Notable:** Deploy animation as hero, terminal aesthetic
- **Patterns:** Gradient mesh backgrounds, code blocks, speed metrics
- **Takeaway:** Show the product, don't describe it

### Stripe.com
- **Notable:** Gradient animations, code-first messaging
- **Patterns:** Interactive demos, bento feature grids, developer docs CTA
- **Takeaway:** Bento grids for features, live code for credibility

### Resend.com
- **Notable:** Developer-first, clean dark theme, email rendering preview
- **Patterns:** Live code examples, minimal sections, fast scroll
- **Takeaway:** Less is more — focus on one killer feature

---

## PATTERNS TO ADOPT

1. **Gradient mesh hero** (Vercel, Stripe) — immersive first impression
2. **Chat/product mockup in hero** (Instantly, Lemlist) — show, don't tell
3. **Social proof bar with logos** (all competitors) — trust signal above fold
4. **Bento feature grid** (Stripe, Linear) — scannable, visual features
5. **Problem/Solution split** (Smartlead) — before/after framing
6. **Animated counters** (Apollo, Smartlead) — metrics create credibility
7. **Scroll-triggered reveals** (Linear, Vercel) — creates discovery momentum
8. **4-column footer** (Stripe, Vercel) — organized, professional

## ANTI-PATTERNS

1. **Wall of text** — no one reads paragraphs on landing pages
2. **Generic stock photos** — screams "we have no product"
3. **Feature dump without hierarchy** — bento grid fixes this
4. **No product preview** — must show the chat interface
5. **Pricing before value** — always earn attention before asking for money
6. **Too many CTAs** — max 2 per viewport (primary + secondary)
