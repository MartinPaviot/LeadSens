# /lp-copy — Landing Page Copywriting Review

You are in **Copywriting** mode. Review and improve all copy on the landing page.

## Process

1. Read ALL section files in `src/app/(marketing)/_sections/`
2. Read `.claude/lp-research/product.md` for positioning
3. Read `.claude/lp-research/brand.md` for voice/tone

## Review Checklist per Section

- [ ] **Headline:** <8 words, benefit-first, no jargon
- [ ] **Subhead:** Supports headline, adds specificity
- [ ] **Body copy:** Scannable, no paragraphs >2 sentences
- [ ] **CTA:** Action verb + outcome (not "Learn More")
- [ ] **Social proof:** Specific numbers over vague claims
- [ ] **Tone:** Confident, not corporate. Technical, not cute.

## Common Copy Fixes

| Problem | Fix |
|---------|-----|
| Feature-first | Rewrite as benefit-first |
| Vague claim | Add specific metric |
| Wall of text | Split into bullet points |
| Weak CTA | "Start for free" > "Get Started" > "Learn More" |
| Corporate tone | Remove "leverage", "utilize", "solution" |

## Rules

- Changes must align with `brand.md` voice
- Preserve technical accuracy (reference `product.md` metrics)
- One section at a time — present before/after for approval
